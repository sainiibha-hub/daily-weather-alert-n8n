# LinkedIn post draft -- n8n Weather Alert Workflow

Built a small automation that's quietly useful: a daily weather alert
workflow in n8n.

How it works:
🕐 Triggers once a day on a schedule
🌤️ Pulls live data from the OpenWeatherMap API
🌧️ Checks if rain is expected — branches the message accordingly
📧📱 Sends the result over both Gmail and Telegram

No backend, no servers to maintain — just nodes wired together in n8n.
A good reminder that not every automation needs a full codebase behind
it.
